发放福利！Linux 和 Windows 下的对拍器模板！

Linux 版本：

#!/bin/bash
		
while true; do
	./data.py > in
	./1 < in > out
	./std < in > ans
	diff -w out ans
	if [ $? = 0 ]; then
		echo 'passed'
	else
		echo 'failed'
		break
	fi
done

Windows 版本：

@echo off
:go
data.py > in.txt
1.exe < in.txt > out.txt
std.exe < in.txt > ans.txt
fc out.txt ans.txt
if not errorlevel 1 goto go
pause
