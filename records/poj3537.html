题目描述

游戏 "Crosses and Crosses" 在一个 $1 \times n$ 的网格上进行，两个人轮流操作。

每次，一个玩家，可以在选择一个空的格子，并在上面标上一个 "×" 号。如果一个玩家操作后网格上有连续 $3$ 个 "×" 号，则她就赢了。

给定 $n$，判断最终谁会赢得该游戏 (假设双方都足够聪明)。

输入格式

共一行，包含一个正整数 $n$ ($3 \leq n \leq 2000$)，表示网格的大小。

输出格式

输出一行一个整数，如果先手赢，则输出 $1$，否则输出 $2$。

题解

代码

#include <bits/stdc++.h>
#define N 2005

int sg[N], mark[50000];

void init() {
	int i, j;
	sg[1] = sg[2] = sg[3] = 1; sg[4] = sg[5] = 2;
	for (i = 6; i < N; ++i) {
		mark[sg[i - 3]] = mark[sg[i - 4]] = mark[sg[i - 5]] = i;
		for (j = 1; j < i - 5; ++j) mark[sg[j] ^ sg[i - j - 5]] = i;
		for (j = 0; mark[j] == i; ++j); sg[i] = j;
	}
}

int main() {
	int n; init();
	return scanf("%d", &n), printf("%d\n", 1 + !sg[n]), 0;
}
